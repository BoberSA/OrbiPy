# OrbiPy
Innovative orbit researching and manipulating tool 

# Info

```bash
Модуль Python для расчета и визуализации орбит в условиях ограниченной круговой задачи трех тел.

Доступные возможности модуля и выражающие их элементы проекта:
Создание модели для решения ограниченной круговой задачи трех тел. По умолчанию доступны системы: Солнце-Земля, Солнце-Венера, Земля-Луна, Юпитер-Ио. (model.py)
Осуществление интегрирования орбит вперед или назад по времени. (integrator.py)
Задание множества "событий" и возможность их детектирования при интегрировании. (events.py)
Осуществление коррекции орбит. (correction.py)
Расчет и визуализация движения космического аппарта по выбранной траектории в течении заданного времени с периодической коррекцией. (station_keeping.py)

Доступен также файл, содержащий информацию о системах (orbi_data.csv), файл с тестами функционала модуля (test.py), файл с реализованным примером использования 
модуля (project_test_1.py).

Спасибо, что выбрали OrbiPy. Летая высоко, чувствуй Землю под ногами.
```
